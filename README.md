# KaleemCoder

A coding-focused LLM built on Qwen3-8B.

## Goals

- Repository understanding
- Bug fixing
- Pull request review
- Code explanation
- Test generation
- Software engineering assistant

## Base Model

Qwen3-8B

## Training

QLoRA + LLaMA Factory

## Hardware

Kaggle Tesla T4 x2

We'll expand this over time.
